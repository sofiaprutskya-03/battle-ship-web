# Морський бій (Battleship Web)

🎮 **Грати онлайн:** [https://sofia-battleship.up.railway.app](https://sofia-battleship.up.railway.app)

## Про гру
Це класична гра "Морський бій" для двох гравців. Реалізована за допомогою сучасних веб-технологій (Node.js, Express, Socket.io) та таблиць WebDataRocks для зручного відображення ігрових полів.

## Правила гри
Гра розрахована строго на двох гравців. 

1. Відкрийте посилання та надішліть його другу.
2. **Розставте 10 кораблів** (сумарно 20 клітинок) на вашому лівому полі:
   - 1 шт. — 4-палубний
   - 2 шт. — 3-палубні
   - 3 шт. — 2-палубні
   - 4 шт. — 1-палубні
3. ⚠️ **Кораблі не можуть торкатися по діагоналі.**
4. Після розстановки натисніть кнопку **"Старт"**.
5. Коли обидва гравці натиснуть "Старт", почнеться гра. Стріляйте по ворожому (правому) полю по черзі, намагаючись потопити всі кораблі супротивника!

**Позначення кольорів:**
- ⬛ **Сірий** — ваш корабель
- 🟥 **Червоний** — влучання у корабель
- ⬜ **Світло-сірий** — промах (постріл у воду)
